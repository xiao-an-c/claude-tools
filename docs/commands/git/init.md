# /git:init

初始化项目 Git 分支规范，创建 master/develop 分支结构。

## 用法

```bash
/git:init
```

## 功能

将现存项目转化为符合 Git 分支规范的格式：

- 检测并确认主分支
- 创建 develop 分支（如果不存在）
- 暂存未提交更改，完成后恢复

## 执行流程

1. 检查是否是 git 仓库
2. 检查未提交更改，有则暂存
3. 检测现有分支，确定主分支名称
4. 创建 develop 分支（如果不存在）
5. 恢复暂存的更改
6. 输出初始化结果

## 主分支检测逻辑

| 情况 | 处理 |
|------|------|
| 已有 master | 使用 master 作为主分支 |
| 已有 main 但无 master | 使用 main 作为主分支 |
| 两者都没有 | 询问用户选择 |
